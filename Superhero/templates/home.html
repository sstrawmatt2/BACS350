Hero Gallery Workshop Blog

This website is the second webiste that I have ever created. My first one was a simple webpage that just allowed for me to show what I can do with basic html, and css code. However, this website is much more sophisticated and allows for the user to do multiple things such as edit heros, delete heroes, wrtie articles about heroes, and so on.

This website is mainly a website to demonstrate the different things that I can do. This was a massive learning curve for me because I have never worked with django before and I've never worked with anything out of my console before. I have learned many things while doing this project such as the importance of inheritance, how to build great templates and views, creating user registration, using html and Python code together, and so much more. I think that this is a great beginner project for new people wanting to get into the field, and you will learn many things!
